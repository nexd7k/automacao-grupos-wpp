# automacao-grupos-wpp
Código Python para automatizar mensagens para grupos de whatsapp usando as bibliotecas openpyxl, pywhatkit e pyautogui.

Para utilizar, o usuário irá precisar do id do grupo do whatsapp que queira mandar mensagem. Para pegar esse ID o usuário precisará ser ADM do grupo, criar um link de compartilhamento e copiar APENAS as letras e numeros que estão localizados após o link https://chat.whatsapp.com/

A automação irá funcionar no navegador padrão, sendo google chrome o navegador recomendado.

A automação sempre irá ser executada no monitor principal e irá controlar seu mouse/teclado. Portanto, é recomendável executar quando precisar fazer alguma outra coisa fora do dispositivo.
